#MonolithCMS ASP Final Version 2005.04


一、新闻系统

50级新闻系统
Html页面静态生成
图片远程保存
新闻审核
新闻栏目合并
新闻二级标题
新闻图片调用

二、广告系统

1级广告分类
支持flash , gif , jpg
统计广告点击率， 包括flash点击计数
广告后加载功能， 加快网页显示
后台更新广告

三、公告系统

自定义显示条数、次数、方式

四、产品/订单管理系统

2级产品分类
自定义交易方式
与会员系统结合
产品可以添加多张产品图片

五、友情连接管理

管理图片以及文字链接
统计各个链接点击数

六、常用小工具（略）

七、管理员工具箱

开放式架构
方便添加、修改、删除个栏目

八、Delphi客户端
提供Delphi客户端源代码，另外附有VB源代码
